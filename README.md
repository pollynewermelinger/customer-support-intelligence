## 🚀 Customer Support Intelligence Platform

Projeto end-to-end de Analytics + IA aplicado a Customer Support, simulando uma operação global de hospitalidade.

### 💡 Principais entregas:
- Pipeline de dados (Python + SQL)
- Dashboard interativo (Streamlit)
- Análise de KPIs (SLA, CSAT, volume)
- Camada de insights automatizados (IA)

### 🎯 Objetivo:
Apoiar tomada de decisão e priorização operacional com base em dados.
